# Tmux 快捷键

<prefix> 开头的是 oh my tmux 设置的快捷键，其他则是原始的快捷键。

# 窗口

c                             创建一个新窗口。
,                             重命名当前窗口。
&                             杀死当前窗口。

0~9                           选择窗口0到9。
<prefix> C-h | C-l            切换窗口
w                             以交互方式选择当前窗口。
<prefix> Tab                  将您带到最后一个活动窗口（两个窗口之间快速切换）
M-p                           移动到带有铃铛或活动标记的上一个窗口。
M-n                           移动到带有铃铛或活动标记的下一个窗口。
p                             切换到上一个窗口。
n                             切换到下一个窗口。
l                             移动到先前选择的窗口。

'                             提示选择窗口索引。
.                             提示索引以移动当前窗口。

Space                         在下一个预设布局中排列当前窗口。

f                             提示在打开的窗口中搜索文本。
i                             显示有关当前窗口的一些信息。

# 窗格

x                             杀死当前窗格。

<prefix> -                    垂直拆分当前窗格
<prefix> _                    水平拆分当前窗格
%                             将当前窗格分成左右两部分。
"                             将当前窗格一分为二，顶部和底部。
<prefix> +                    将当前窗格最大化到新窗口
z                             切换当前窗格的缩放状态。

q                             简要显示窗格索引。
;                             移至先前活动的窗格(两个窗格之间快速切换)。
○                             在当前窗口中选择下一个窗格。
<prefix> h j k l              让您在Vim中导航窗格
Up，Down，Left，Right         切换到当前窗格上方、下方、左侧或右侧的窗格。

<prefix> H J K L              让您调整窗格大小
C-Up，C-Down，C-Left，C-Right 以一个单元格为单位调整当前窗格的大小。
M-Up，M-Down，M-Left，M-Right 以五个单元格为单位调整当前窗格的大小。

{                             将当前窗格与前一个窗格交换。
}                             将当前窗格与下一个窗格交换。
C-o                           向前旋转当前窗口中的窗格。
M-o                           向后旋转当前窗口中的窗格。
<prefix> < >                  交换窗格（交换位置）

M-1~M-5                       以五种预设布局之一排列窗格：偶数水平、偶数垂直、主水平、主垂直或平铺。

!                             将当前窗格转换成一个新的窗口。

m                             标记当前窗格（请参阅select-pane_-m）。
M                             清除标记的窗格。

# 会话

<prefix> C-c                  创建一个新会话
$                             重命名当前会话。

L                             将附加的客户端切换回上一个会话。
(                             将附加的客户端切换到上一个会话。
)                             将附加的客户端切换到下一个会话。
<prefix> C-f                  让您按名称切换到另一个会话

s                             以交互方式为附加的客户端选择一个新会话。

# 客户端

d                             分离当前客户端。
D                             选择要分离的客户端。
r                             强制重绘附加的客户端。
C-z                           暂停tmux客户端。

# 其他

<prefix> b | #                列出粘贴缓冲区
<prefix> p                    来自顶部粘贴缓冲区的粘贴
<prefix> P                    让您选择要粘贴的粘贴缓冲区
=                             从列表中选择要以交互方式粘贴的缓冲区。
-                             删除最近复制的文本缓冲区。

<prefix> Enter                进入复制模式
[                             进入复制模式以复制文本或查看历史记录。

]                             粘贴最近复制的文本缓冲区。

:                             输入tmux命令提示符。
?                             列出所有键绑定。
t                             显示时间。
~                             显示以前的消息tmux，如果有的话。
PageUp                        进入复印模式并向上滚动一页。
<prefix> e                    使用环境变量~/.tmux.conf.local定义的编辑器打开（默认为空时）$EDITORvim
<prefix> r                    重新加载配置
C-l                           清除屏幕和tmux历史记录
<prefix> m                    打开或关闭鼠标模式
<prefix> U                    启动Urlview（如果可用）
<prefix> F                    启动Facebook_PathPicker（如果可用）
